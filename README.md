# FRESHMART LOCAL GROCERY STORE WEBSITE

## Project Overview
This project is a static website for **FreshMart Local Grocery Store**. It provides store information, a product catalog, and an online enquiry page so customers can learn about the business and contact the store.

## Website Goals & Objectives
- **Inform visitors** about FreshMart and what the store offers.
- **Showcase products** through a clear products catalog.
- **Enable customer communication** via an enquiry form and contact details.
- **Improve usability** with consistent navigation, readable layout, and responsive design.

## Key Features & Functionality
- **Home (index.html):** Hero banner + store introduction and quick marketing sections.
- **Products (products.html):** Product catalog grid/cards with images and “Buy Now” buttons (front-end only).
- **About Us (about.html):** Store background, mission-style description, and service highlights.
- **Contact (contact.html):** Contact information and a message form (front-end only).
- **Enquiry (enquiry.html):** Product enquiry form for customers to send questions/suggestions (front-end only).
- **Shared UI elements:**
  - Consistent header/navigation across pages.
  - Consistent multi-section footer including social links and contact details.
  - Font Awesome icons for social/contact visuals.

## Site Map

## Changelog (Website updates)

- **Products catalog (products.html):** Created a product grid/card layout with product images, product names, prices, and “Buy Now” buttons (UI/front-end).
- **About page (about.html):** Added the store story/values and a “Why choose us” section highlighting delivery, fresh produce, affordability, and customer satisfaction.
- **Contact page (contact.html):** Added a contact information section (phone, email, physical address) and a front-end message form.
- **Enquiry page (enquiry.html):** Added an enquiry form for product-related questions, feedback, and suggestions (front-end).
- **Assets integration:** Included images and media from the `assets/images/` and `assets/media/` folders to support the content and product catalogue.
- **Branding:** Integrated the project logo in the header to keep the site visually consistent.
- **Usability improvements:** Ensured pages use the shared stylesheet (`assets/css/style.css`) for consistent typography, spacing, and responsive layout.
